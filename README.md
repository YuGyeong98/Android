## 🛠 Features

1. 단위 변환 연산
    - cm → m
    - m → cm
2. scale(자릿수)에 따라 화면에 표시되는 텍스트 변환
    - 소수점 6자리까지만 표현
3. 사용자가 텍스트 입력 시, 화면을 초과하는 경우 방지
