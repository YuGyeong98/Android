## 🛠 Features

1. 지역을 검색하면 맛집 리스트 나타내기
2. 리스트 중 하나를 눌렀을 때, 그 위치로 지도 및 카메라 이동
3. 마커를 눌렀을 때, 해당 맛집 BottomSheet 나타내기
4. 맛집 링크 클릭하면 해당 페이지로 이동

## ✏️ Skills

* Naver Map
* Retrofit
  * Moshi
* BottomSheet
* Toolbar
* SearchView
* Parcelize
