# 음악 플레이어 앱

## 🛠 Features

1. 음악 재생, 일시정지, 정지
2. 음악 재생되면 Notification 생성
3. LOW_BATTERY 이벤트 캐치하여 Toast 띄우기

## ✏️ Skills

* MediaPlayer
* Service
* Notification
* BroadcastReceiver
* IntentFilter
* PendingIntent

## 🐣 Challenges
- [ ] 음악 재생되는 동안 Notification 제거 불가
- [ ] 음악 정지 후에는 Notification 제거 가능
